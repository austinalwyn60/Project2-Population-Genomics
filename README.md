# Population Genomics - Pharmacogenomic Variants Analysis

## Project Title
Population-Stratified Allele Frequency Analysis of Pharmacogenomic Variants Across South Asian and European Populations

**Student:** Austin d Almeida  
**Date:** 26 June 2026

## Objective
To analyze allele frequency differences of 10 clinically important pharmacogenomic variants between South Asian (SAS) and European (EUR) populations and discuss implications for Indian patients.

## Key Files
- `results/frequency_data.csv` - Frequency data for 10 variants
- `results/comparison.txt` - Summary of key differences
- `results/FINAL_REPORT.txt` - Complete written report
- `plots/sas_vs_eur.png` - Main result plot (SAS vs EUR)
- `plots/frequency_comparison.png` - All populations comparison
- `scripts/visualize.R` - R script to generate plots

## Major Findings
- CYP2C19 variants are significantly more common in South Asians (2.3x higher)
- SLCO1B1 and IFNL3 also show notable differences
- Important clinical implications for drug dosing in Indian patients

## Tools Used
- R + ggplot2
- 1000 Genomes Phase 3 Data

## Conclusion
This project highlights the need for population-specific pharmacogenomic guidelines in India.
